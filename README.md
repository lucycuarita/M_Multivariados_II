#  METODOS MULTIVARIADOS II (EST-337)

## `Objetivo General`
> Aplicar los conocimientos del Álgebra Lineal y la asignatura de Métodos Multivariados I en el desarrollo analítico y aplicativo de técnicas multivariantes, a un nivel productivo-creativo.
